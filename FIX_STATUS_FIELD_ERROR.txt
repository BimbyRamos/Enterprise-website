========================================
FIX: "Status" Field Name Error
========================================

ERROR MESSAGE:
"This name cannot be used in your content type as it might break other functionalities"

PROBLEM:
"status" is a RESERVED WORD in Strapi and cannot be used as a field name.

SOLUTION:
Use "projectStatus" instead of "status"

========================================
OTHER RESERVED WORDS TO AVOID
========================================

Don't use these as field names in Strapi:
- status
- id
- createdAt
- updatedAt
- publishedAt
- createdBy
- updatedBy
- locale
- localizations
